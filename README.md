# Modul1 - Demo UI Mobile Banking BNI

Aplikasi Android demo UI Mobile Banking BNI - tugas kampus.

## Fitur
- Login & Register (Firebase Auth)
- Forgot & Reset Password
- Home Dashboard (Saldo + 6 menu layanan)
- Transfer, QRIS, E-Wallet, Pembayaran, Pembelian
- Mutasi & Detail Transaksi (RecyclerView)
- Profile
- Layanan (Implicit Intent): Maps, Kamera, Share, Website, Telepon, YouTube, Instagram, Berita
- Setting Dialog (Ubah Bahasa, Notifikasi, Keamanan, Tentang)

## Tech
Kotlin · XML · ViewBinding · Firebase Auth · SharedPreferences · Material Design 3

## Build
```bash
./gradlew assembleDebug
./gradlew installDebug
```
